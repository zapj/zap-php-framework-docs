# 安装

```bash
composer require zapj/zap-php-framework
```


## 目录文件

```
├─ app
│  ├─ zap
│  │  └─ controllers
│  └─ mods
├─ config
│  ├─ database.php  #数据库配置
│  └─ config.php    #配置文件
├─ storage
├─ var
├─ bin
├─ .gitignore
├─ index.php
└─ composer.json
```

## Console

```
php bin/console DB@create
```
